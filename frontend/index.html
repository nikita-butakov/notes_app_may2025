    Add Note

    Created by Nikita Butakov | Release: 25 May 2025

    This is a simple production-style Notes App built with:

    • Flask — Python backend framework for handling API routes
    • Docker — containerizing both frontend and backend for portability
    • Amazon ECS (Fargate) — serverless container hosting without managing EC2
    • Amazon RDS (PostgreSQL) — managed database for note storage
    • Amazon CloudWatch — centralized logging and monitoring (optional)
    • GitHub Actions — CI/CD pipeline for building and pushing Docker images to
    • Amazon ECR — container registry used to store application images
